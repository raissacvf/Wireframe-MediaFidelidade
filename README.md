# Wireframe de Média Fidelidade - Sistema de Agendamento para Salão de Beleza

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte do desafio da DIO, com o objetivo de criar um wireframe de média fidelidade para um sistema de agendamento de salão de beleza.

A proposta foi desenvolver uma interface simples e intuitiva, permitindo que o usuário realize o agendamento de um serviço em poucos passos.

---

## 🎯 Objetivos

- Criar um wireframe de média fidelidade;
- Aplicar conceitos de UX (User Experience);
- Utilizar princípios de usabilidade e psicologia do design;
- Desenvolver uma interface organizada e de fácil navegação.

---

## 🛠 Ferramenta utilizada

**Bootstrap**

### Por que escolhi o Bootstrap?

Escolhi utilizar o Bootstrap por ser uma ferramenta que já domino e que permite criar protótipos de forma rápida e organizada. Com ela foi possível estruturar a interface e validar o fluxo de navegação de maneira eficiente, mantendo o foco nos conceitos de usabilidade e experiência do usuário propostos pelo desafio.

---

## 📱 Fluxo do usuário

O processo de agendamento foi dividido em três etapas:

1. Seleção do serviço;
2. Escolha da data e horário;
3. Confirmação do agendamento.

O objetivo foi criar um fluxo simples, reduzindo a quantidade de ações necessárias para concluir a tarefa.

---

## 🎨 Conceitos aplicados

- Hierarquia visual;
- Consistência entre as telas;
- Simplicidade;
- Feedback ao usuário;
- Redução da carga cognitiva;
- Navegação intuitiva.
